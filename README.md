# 🔗 Shortly - URL Shortening Landing Page

A modern, fully responsive landing page for a URL shortening service. Built with **HTML5**, **Tailwind CSS**, and **Vanilla JavaScript**, this project showcases responsive design, client-side form validation, and an engaging user interface with interactive components.

## ✨ Features

- 📱 Fully responsive design for desktop, tablet, and mobile devices
- 🔗 Client-side URL validation using JavaScript Regular Expressions
- ❌ Displays error messages and input validation states for invalid URLs
- 🍔 Responsive mobile navigation with hamburger menu
- 🎨 Modern UI built with Tailwind CSS
- ✨ Interactive buttons and hover animations
- 📐 Responsive layouts using Flexbox and CSS Grid
- ⚡ Fast-loading static website
- 🚀 Ready for deployment on Netlify, Vercel, or GitHub Pages

## 🚀 Getting Started

### 1. Clone the Repository

```bash
git clone https://github.com/Rizwan-46/shortly-tailwind-css.git

cd shortly-tailwind-css
```

### 2. Install Dependencies

```bash
npm install
```

### 3. Start Tailwind CSS Watch Mode

```bash
npx tailwindcss -i ./input.css -o ./output.css --watch
```

If your `package.json` includes custom scripts, you can also run:

```bash
npm run dev
```

or

```bash
npm run watch
```

### 4. Open the Project

Open `index.html` directly in your browser or use the **Live Server** extension in Visual Studio Code for automatic reloading during development.
